Contrast-Preserving Hardware Dimming 대비를 보존하는 하드웨어 감마 디밍

Unlike standard dimming tools that cover your screen with a transparent black window, Whiteout modifies the hardware gamma lookup tables inside your Mac's graphics card. 화면 전체에 어두운 창을 씌워 시야를 가리는 일반 필터 소프트웨어와 달리, 화이트아웃은 맥 그래픽 카드 내부의 하드웨어 수준 감마 테이블을 직접 제어합니다.

Traditional Software Overlays 일반 소프트웨어 오버레이

Ruins Contrast & Blacks 대비 손상 및 들뜨는 블랙

Puts a semi-transparent black overlay window over the screen. Pure black gets elevated to a muddy grey, and whites become dark grey. Colors look washed out, and reading text in low light becomes strainful due to crushed contrast. 화면 위에 검은색 반투명 레이어를 덮는 방식입니다. 검은색 텍스트와 배경이 회색조로 뿌옇게 들뜨고, 명암비가 급격히 손실되어 저조도 환경에서 글자를 장시간 읽을 때 오히려 안구 피로를 가중시킵니다.

Preserves Contrast & Darks 명암비 보존 및 트루 블랙 유지

Modifies the GPU's transfer function. Pure black stays absolute black (0.0). High brightness levels are compressed down using a customizable mathematical curve. Text remains crisp and black, preserving readability. 그래픽 카드의 신호 레벨 자체를 조절합니다. 완벽한 블랙(0.0) 신호는 그대로 두고 밝은 화이트 광원 영역만 부드럽게 감쇄합니다. 명암대비가 보존되어 텍스트가 칼같이 선명하게 유지되며 눈에 편안합니다.

The Non-Linear Brightness Curve 비선형 밝기 감쇄 곡선

Whiteout uses a power-based exponent curve to map input pixel values from the GPU to the display. This ensures that dark and mid-tone values are compressed much less than peak highlights. 화이트아웃은 화면 신호 변환 시 단순 1차원 선형 필터를 사용하지 않고 수학적 비선형 지수 공식을 적용합니다. 어둡거나 평범한 중간 밝기 영역은 원본에 최대한 보존하여 왜곡을 없애고, 순백색을 비롯한 극단적인 눈부심 유발 영역만 효율적으로 잡아줍니다.

Three Curve Exponent Presets 목적에 맞춰 세분화된 세 가지 프리셋

T = 2.5
General Mode 일반 모드 Smooth, balanced reduction. Best for daily activities, browsing web pages, and light coding. 전반적으로 고르고 부드럽게 밝기를 낮춥니다. 웹서핑, 오피스 작업 등 일상적인 웹 활용 환경에 가장 무난한 추천 기본 모드입니다.
T = 4.0
Document & PDF Mode 문서·PDF 모드 Concentrated white compression. Maximizes black text contrast while muting bright white page backgrounds, making PDF reading exceptionally comfortable. 흰색 배경 영역만 집중적으로 강하게 압축합니다. 검정색 텍스트와 라인의 선명도는 100% 살린 채, 눈을 시리게 만드는 문서 배경을 부드럽게 가라앉혀 줍니다.
T = 6.0
Highlights-Only Mode 하이라이트 집중 모드 Keeps darks and mid-tones almost entirely untouched. Targets only the brightest pure highlights, ideal for viewing dark content in a pitch-black room. 중간 톤을 거의 완벽히 유지하고 극단적인 순백색(Pure White) 부분만 눌러줍니다. 어두운 배경 위주인 소스 코드 에디터 활용이나 불 꺼진 침실 작업 시 탁월합니다.

Designed for Power Users 맥 유저를 위한 세심한 설계

Whiteout packs advanced features in a lightweight menu bar app to deliver a seamless, native macOS experience. 화이트아웃은 가볍게 동작하는 메뉴 막대 전용 유틸리티로, 맥OS 고유의 심플하고 군더더기 없는 사용성을 제공합니다.

Multi-Monitor Support 다중 모니터 동시 제어

Applies white point reduction to all active displays. Dynamically adapts when plugging or unplugging monitors without needing restarts. 연결된 모든 모니터의 화이트포인트를 일괄 디밍합니다. 시스템 수준에서 모니터 연결이 감지(핫플러그)되면 프로그램 재시작 없이 즉시 감마 테이블을 동기화합니다.

Customizable Shortcuts 사용자 설정 글로벌 단축키

Register system-wide keyboard shortcuts. Toggle Whiteout on and off instantly from any active application with smooth fade transitions. 작업 도중 언제 어디서나 단축키 조작 한 번으로 디밍 효과를 켜고 끌 수 있습니다. macOS 표준 API 기반의 직관적인 단축키 녹화기를 지원합니다.

Contrast Reversal Safeguard 명암비 컬러 역전 방지 장치

Restricts reduction between 0% and 30% with tactile 5% snaps. This prevents visual artifacts and color inversion typical at higher ranges. 디밍 범위를 검증된 수준인 0~30%로 제한하고 5% 마다 부드럽게 감기는 스냅을 제공합니다. 이는 너무 어둡게 설정되어 색상이 일그러지거나 대비가 무너지는 왜곡을 원천 봉쇄합니다.

100% Native & Lightweight 가볍고 쾌적한 네이티브 설계

Built completely in Swift using Cocoa & SwiftUI APIs. Runs directly in the menu bar. Consumes less than 15MB RAM and 0% idle CPU. Swift 언어와 오리지널 Cocoa/SwiftUI 프레임워크로 제작되었습니다. 메모리 점유율 15MB 이하, 유휴 상태일 때 CPU 소모량 0%를 유지해 성능 저하가 없습니다.

Safe Hardware Restores 안전한 종료 감마 복구

Listens to app terminate notifications and system changes. Safely restores the original factory display gamma tables on app exit. 앱 종료(`cmd+Q`)를 감지하거나 사용자가 초기화 버튼을 누르면, 디스플레이 하드웨어 테이블 설정을 깨끗하게 초기 오리지널 값으로 원복하여 뒤끝을 남기지 않습니다.

Seamless Auto Updates 배너 원클릭 자동 업데이트

Automatically checks for GitHub releases. Download, install, and relaunch into the newest version silently with a single click. 깃허브 API를 모니터링하여 새 버전이 릴리즈되면 알아서 UI에 업데이트를 표시합니다. 복잡한 설치 과정 없이 배너를 클릭하면 백그라운드 갱신 및 재시작까지 알아서 진행됩니다.

Gatekeeper Bypass 보안 경고 해결 방법

⚠️ Security & Quarantine Warning (Gatekeeper) ⚠️ macOS 보안 격리(Gatekeeper) 해결법

Because Whiteout is self-signed and distributed off-App Store, macOS may alert that the file is damaged or cannot be checked. 화이트아웃은 인디 서명으로 배포되는 오픈소스 외부 프로그램입니다. 다운로드 후 실행이 안 되거나 '손상된 파일'이라는 문구가 뜰 시 아래 방법으로 간단히 해제할 수 있습니다.

Method 1 (Simple):방법 1 (간단): Right-click Whiteout.app in your Applications folder and click Open twice. 응용 프로그램 폴더에 넣은 Whiteout.app 파일을 마우스 우클릭한 뒤 열기 -> 열기를 차례대로 누릅니다.

or

Method 2 (Terminal):방법 2 (터미널): Run the following command in terminal to clear the quarantine flag: 터미널 앱을 열고 아래 명령어를 복사하여 입력해 격리 캐시를 해제해 줍니다: 

xattr -dr com.apple.quarantine /Applications/Whiteout.app
