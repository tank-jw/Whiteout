Contrast-Preserved GPU Dimming

Unlike standard dimming tools that cover your screen with a transparent black window, Whiteout modifies the hardware gamma lookup tables inside your Mac's graphics card.

The Non-Linear Brightness Curve

Whiteout uses a power-based exponent curve to map input pixel values from the GPU to the display. This ensures that dark and mid-tone values are compressed much less than peak highlights.

Three Curve Exponent Presets

T = 2.5
General Mode: Smooth, balanced reduction. Best for daily activities, browsing web pages, and light coding.
T = 4.0
Document & PDF Mode: Concentrated white compression. Maximizes black text contrast while muting bright white page backgrounds, making PDF reading exceptionally comfortable.
T = 6.0
Highlights-Only Mode: Keeps darks and mid-tones almost entirely untouched. Targets only the brightest pure highlights, ideal for viewing dark content in a pitch-black room.

Designed for Power Users

Whiteout packs advanced features in a lightweight menu bar app to deliver a seamless, native macOS experience.

Multi-Monitor Support

Applies white point reduction to all active displays. Dynamically adapts when plugging or unplugging monitors without needing restarts.

Customizable Shortcuts

Register system-wide keyboard shortcuts. Toggle Whiteout on and off instantly from any active application with smooth fade transitions.

Contrast Reversal Safeguard

Restricts reduction between 0% and 30% with tactile 5% snaps. This prevents visual artifacts and color inversion typical at higher ranges.

100% Native & Lightweight

Built completely in Swift using Cocoa & SwiftUI APIs. Runs directly in the menu bar. Consumes less than 15MB RAM and 0% idle CPU.

Safe Hardware Restores

Listens to app terminate notifications and system changes. Safely restores the original factory display gamma tables on app exit.

Seamless Auto Updates

Automatically checks for GitHub releases. Download, install, and relaunch into the newest version silently with a single click.

Troubleshooting

Gatekeeper Bypass

⚠️ Security & Quarantine Warning (Gatekeeper)

Because Whiteout is self-signed and distributed off-App Store, macOS may alert that the file is damaged or cannot be checked.

Method 1 (Simple): Right-click Whiteout.app in your Applications folder and click Open twice.

or

Method 2 (Terminal): Run the following command in terminal to clear the quarantine flag:

xattr -dr com.apple.quarantine /Applications/Whiteout.app
